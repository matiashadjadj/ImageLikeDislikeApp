# Dating App

## Overview

The Dating App is a simple graphical user interface (GUI) application built with Python using Tkinter. It allows users to view images and record likes and dislikes for each image. This application was developed for a research project to track user preferences regarding images. The goal of this app is not to match different users together but simply to record the preference of users.

## Features

- Import images from any folder on the user's computer.
- View images one at a time with options to like or dislike each image.
- Log user actions in an action logger window that tracks the total number of likes and dislikes.
- Reset the logged actions at any time.
- Supports dynamic resizing of images while maintaining their aspect ratio.
